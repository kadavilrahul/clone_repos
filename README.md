# Repository Analysis Toolkit

A comprehensive collection of bash scripts for analyzing, comparing, and managing code repositories.

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/kadavilrahul/clone_repos.git
```
```bash
cd clone_repos
```


# Run interactive analysis
```bash
bash report.sh
```

# Compare multiple codebases
```bash
bash compare.sh
```

# Clone GitHub repositories
```bash
bash clone.sh
```

## 📋 Scripts Overview

### 🔍 `report.sh` - Comprehensive Project Analysis
**The flagship analysis tool** with interactive navigation and detailed reporting.

**Features:**
- 📊 Project structure visualization
- 🔒 Security pattern scanning
- 📈 Code statistics by language
- 🔍 Function extraction
- 📝 Git repository statistics
- 🎨 Colorized terminal output
- 📁 Auto-opens report folder

**Usage:**
```bash
# Interactive mode
./report.sh

# Direct analysis
./report.sh /path/to/project report

# Extract functions
./report.sh /path/to/project functions

# Search for patterns
./report.sh /path/to/project search "TODO"

# Git statistics
./report.sh /path/to/project git
```

### 📊 `compare.sh` - Multi-Codebase Line Counter
Compare total lines of code across multiple projects with enhanced error handling.

**Features:**
- 📏 Accurate line counting with file statistics
- 🛡️ Robust error handling and validation
- 🏠 Home directory expansion (`~` support)
- 📝 Detailed output with file counts
- 💾 Results saved to timestamped files

**Usage:**
```bash
./compare.sh
# Follow interactive prompts to add multiple codebases
```

### 🔄 `clone.sh` - GitHub Repository Manager
Advanced GitHub repository cloning with favorites, filtering, and batch operations.

**Features:**
- 🔐 GitHub API integration
- ⭐ Favorites management
- 🔍 Repository filtering (public/private, languages, forks)
- 📚 Recent repositories tracking
- 🎯 Interactive repository selection

**Usage:**
```bash
./clone.sh
# Follow interactive prompts for GitHub authentication and repository selection
```

### Initial Setup

1. **Configure GitHub Access** (for clone.sh):
   ```bash
   # Option 1: Environment variable (recommended)
   export GITHUB_TOKEN="your_github_token_here"
   
   # Option 2: Edit config.json (not recommended for security)
   # See config.json for structure
   ```

2. **Install Optional Dependencies**:
   ```bash

   sudo apt install tree cloc fzf jq
   
   # Or on macOS
   ```

## 📁 Output Structure

```
clone_repos/
├── reports/                    # Generated by report.sh
│   ├── project_report_*.md    # Project analysis reports
│   ├── cloc_*.txt            # CLOC output files
│   └── functions_*.txt       # Function extraction results
├── codebase_line_counts.txt   # Generated by compare.sh
└── repos/                     # Default clone location
```

## 🔧 Dependencies

### Required
- `bash` (4.0+)
- `find`, `wc`, `grep` (standard Unix tools)
- `git` (for clone.sh and git statistics)

### Optional (Enhanced Features)
- `tree` - Better project structure visualization
- `cloc` - Detailed code counting
- `jq` - JSON parsing for configuration
- `fzf` - Enhanced interactive selection
- `rg` (ripgrep) - Faster searching

### Installation Commands
```bash
# Ubuntu/Debian
sudo apt update && sudo apt install tree cloc fzf jq

# CentOS/RHEL
sudo yum install tree cloc fzf jq

# macOS
brew install tree cloc fzf jq

# Arch Linux
sudo pacman -S tree cloc fzf jq
```

## 🔒 Security Notes

⚠️ **Important**: Never commit GitHub tokens to version control.

**Secure Token Management:**
```bash
# Method 1: Environment variable
echo 'export GITHUB_TOKEN="your_token"' >> ~/.bashrc
source ~/.bashrc

# Method 2: Secure file
echo "your_token" > ~/.github_token
chmod 600 ~/.github_token
```

## 🐛 Troubleshooting

### Common Issues

1. **Permission Denied**
   ```bash
   chmod +x *.sh
   ```

2. **GitHub API Rate Limiting**
   - Use authenticated requests with token
   - Wait for rate limit reset

3. **Missing Dependencies**
   ```bash
   # Check what's missing
   ./report.sh help
   # Install missing tools as shown above
   ```

4. **Path Issues**
   - Use absolute paths when in doubt
   - Check directory permissions
